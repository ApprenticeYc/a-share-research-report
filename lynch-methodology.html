彼得·林奇投资体系

可复用的投研方法论提炼自《彼得·林奇的成功投资》

从投资边界、线索发现、公司分类、质量筛选，到估值核查、买入故事、持续追踪和卖出换仓。

一套用于个股研究的操作框架：先确定投资边界，再发现可验证的公司变化，随后分类、筛选、估值、核查、买入、追踪、卖出。

核心主线：

投资边界 → 发现线索 → 判断公司类型 → 验证质量与估值 → 讲清公司逻辑 → 买后持续追踪 → 卖出或换仓。

这套方法可以分成三层。第一层是投资边界，解决钱、能力圈和市场噪音；第二层是投研决策，解决线索、分类、筛选、估值和买入；第三层是持有复盘，解决买入后的验证、迁移、卖出和换仓。

层级作用对应动作
投资边界决定哪些钱能投、哪些公司能看、哪些噪音要过滤只用长期闲钱；只研究能力圈；不把短期市场预测当买卖理由
投研决策决定一家公司是否值得买发现线索、公司分类、质量筛选、财务估值、信息核查、买入故事
持有复盘决定收益能否兑现定期复查、识别类型迁移、判断故事增强或破裂、卖出或换仓

读完后要形成的不是几个概念，而是一套动作：每分析一家公司，都能知道下一步该查什么、什么时候停止、什么时候等待、什么时候卖出。

0. 总览：从发现线索到卖出换仓

林奇的方法不是单点选股技巧，而是一条完整的投研链条：

从真实世界发现线索，用公司类型提出正确问题，用正反筛选排除噪音，用财务和估值约束故事，用两分钟投资故事检验自己是否真正看懂公司，用持续追踪决定持有、加仓、减仓、卖出。

0.1 八条核心原则

#原则实操含义
1只买自己能理解的公司讲不清业务、利润来源、关键变量，就不进入研究
2身边观察是线索，不是买入理由看到排队、复购、订单改善后，还要验证重要性和估值
3买入前先分类六类公司对应六套问题，分类错后面全错
4价格必须与收益、增长、资产匹配好公司买贵了也会变成坏股票
5最好的机会常在冷门、枯燥、被忽视的公司里超额收益来自市场还没充分定价
6最大风险常在热门、昂贵、故事动听但缺利润的公司里故事不能替代收益、现金和负债
7长期投资不是不动，而是持续验证基本面持有是复查循环，不是买完忘记
8卖出理由来自买入理由失效，不来自股价本身买入写故事，卖出查故事

0.2 七步闭环

步骤核心问题结果
1. 发现我看到了什么可验证的公司变化？候选公司
2. 分类它属于六类中的哪一类？研究题型
3. 筛选它符合正面线索吗？踩中排除对象吗？进入深研 / 观察 / 排除
4. 估值当前价格要求公司未来做到什么？估值压力测试
5. 决策两三分钟能否讲清公司如何赚钱、为什么现在值得买、什么情况会卖？买入 / 不买
6. 持有故事变强、变弱，还是类型迁移？持有 / 加仓 / 减仓
7. 卖出故事破裂、估值透支，还是有更好机会？卖出 / 换仓

这个闭环的价值不是让你一次性做完所有动作，而是让每次研究都能回到同一套问题：发现是否真实，分类是否正确，数字是否支持，买入理由是否还能成立。

这个闭环最重要的四个箭头：

  1. 决策不通过 → 回到发现。 大多数公司被淘汰不是失败，而是纪律。
  2. 持有 → 持有。 好公司不是每天交易，而是持续验证。
  3. 卖出 → 重新发现。 卖出不是结束，是资本重新寻找更好故事。
  4. 心理护盾保护全流程。 12种危险说法、期权期货卖空、宏观恐慌，都会把你从企业研究拉回价格赌博。

1. 投资前提：什么人适合主动投资

对应原书主题：第1-5章，尤其是“进入股市前的自我测试”和“不要预测股市”。

这一章不需要反复深挖，但不能省略。它解决的不是“买哪家公司”，而是“你有没有资格主动选股”。

1.1 主动投资前的五条边界

前提林奇的意思实操翻译
长期闲钱股票波动不可避免，短钱会被迫在错误时间卖出生活资金、买房资金、短期负债资金不能用于主动选股
能力圈不买自己讲不清楚的公司能解释业务、利润来源、关键变量，才进入研究
不预测短期市场没人能稳定预测大盘、利率、经济衰退宏观只能翻译成公司变量，不能作为清仓/满仓按钮
独立判断机构和专家也会错，热门观点常滞后所有推荐都拆成可验证假设
情绪防线股价涨跌会覆盖事实判断买入前写下理由，卖出时只检查理由是否失效

1.2 业余投资者的优势不是“更聪明”

林奇说业余投资者可能战胜机构，不是因为业余投资者天然聪明，而是因为结构不同。

对比维度业余投资者专业投资者
信息来源工作、消费、生活观察，可能领先6-12个月分析师报告、会议、共识信息，常常滞后
投资限制可以不买，可以只买懂的公司常被名单、规模、风控、职业风险限制
资金规模小资金灵活，能买冷门小公司基金太大，小公司对业绩贡献有限
职业压力只对自己负责买冷门错了会被质疑，买共识错了较安全
时间自由可以等待几年常受季度排名和赎回压力影响

这不是鼓励盲目自信，而是提醒：你的优势只存在于“你先看到真实经营变化，并且比市场更早完成验证”的时候。

1.3 不适合主动投资的情况

  1. 资金两三年内必须使用。
  2. 无法承受股价下跌，跌了就想止损。
  3. 不愿意读财报、公告、电话会、行业资料。
  4. 喜欢听代码和结论，不愿意写自己的买入理由。
  5. 讲不清公司靠什么赚钱，却愿意重仓。
  6. 容易把宏观新闻、股价走势、别人观点当作买卖理由。

1.4 适合主动投资的情况

  1. 有长期闲钱。
  2. 有生活、行业或专业场景能持续观察。
  3. 愿意把线索转化成事实验证。
  4. 能接受大量时间只是等待和复查。
  5. 能承认错误，故事破裂时卖出。

2. 发现线索：识别可验证的公司变化

对应原书主题：第6章“寻找10倍股”，以及导论中“业余投资者的优势”。

“从生活中找机会”不是看到排队就买，也不是产品好就买。正确说法是：

你看到了什么具体变化？如果这个变化成立，它会怎样改善公司的收入、利润、现金流或资产价值？

现实观察只是入口，必须翻译成可验证的经营判断。

2.1 四类线索来源

线索来源观察到的变化需要验证的问题第一轮核查
日常消费门店排队、复购强、体验好、家人朋友反馈好这是单点现象，还是可复制需求？产品/门店收入占比、同店销售、毛利率、开店速度
工作行业订单增加、库存下降、价格上涨、客户转向某供应商这是短期波动，还是公司竞争位置改善？订单、价格、库存、产能、客户集中度、成本位置
地方资产本地地产、区域银行、地方品牌、牌照资源被低估市场是否没看见资产价值？价值能否释放？资产可变现价值、债务扣除、催化剂、管理层行为
专业接触医院药品、供应链、渠道、工厂、客户选择变化现场变化能否进入财务报表？收入确认、利润率、客户留存、替代风险、监管风险

2.2 形式过时，本质不过时

林奇书里有一些方法具有时代特征，例如给公司打电话、拜访总部、逛商场、问经纪人。今天不应机械照搬，但本质没有过时。

林奇原方法今天可能过时的地方不过时的本质今天可替代的方法
逛商店、看门店线下零售不覆盖所有商业模式到真实交易现场观察需求门店客流、线上评价、App排名、渠道调研、复购数据
向身边人了解产品样本小，容易偏从真实用户行为找线索用户访谈、社群反馈、退款率、留存率、搜索热度
给公司打电话普通投资者未必能获得增量信息，也有合规边界主动核查关键假设财报电话会、业绩说明会、公告问答、投资者纪要
拜访总部成本高，容易看到公司展示面观察管理层风格和资本纪律历史并购、回购、分红、增发、薪酬激励、关联交易
阅读年报信息滞后用事实约束故事年报、季报、公告、监管文件、同业对比、招股书

2.3 产品好、公司好、股票好，是三件不同的事

层级说明必问问题
好产品只是线索产品占公司收入和利润多少？
好公司商业模式可能成立成功能否复制？利润率和现金流是否好？
好股票价格仍有吸引力估值是否已经透支好消息？

一个消费线索必须先过六个问题：

  1. 这个产品或服务占公司收入多少？
  2. 它占利润多少？
  3. 毛利率和现金流是否好？
  4. 是一次性爆品，还是反复购买？
  5. 成功能否跨门店、跨地区、跨渠道复制？
  6. 当前估值是否已经反映完好消息？

如果产品占比很小，它只是新闻，不是投资理由。如果模式不能复制，它只是好体验，不是成长股。如果估值已经很高，它可能是好公司，但不是好股票。

2.4 关键案例：La Quinta 与 Bildner's

案例发现方式关键验证结果教训
La Quinta问行业人士“最强对手是谁”客房质量接近假日酒店，价格低约30%，砍掉会议室/餐厅，专注小商人，模型可复制成为大牛股好线索必须转化成可复制的经营模型
Bildner's办公室对面三明治好吃没有充分验证离开本地后能否复制，扩张失败股价大跌好产品不等于好股票，单店体验不能替代商业模式

3. 公司定性：六类公司决定六种研究问题

对应原书主题：第7章“6种类型公司股票”。

林奇的六类公司不是标签，而是问题生成器。分类一错，估值、追踪和卖出都会错。

3.1 分类第一问：股东收益主要来自哪里？

判断公司类型时，不要先看行业名称，也不要先看股价涨跌。先问：这笔投资的收益主要从哪里来？

第一判断更可能的类型继续问
收益主要来自稳定股息，增长很慢缓慢增长型股息安全吗？现金流能否覆盖？上涨30%-50%后还有吸引力吗？
公司已经很大，收益稳健增长，目标是阶段性稳健回报稳定增长型PE是否合理？增长是否还能保持？衰退时会不会明显受损？
公司规模仍不大，盈利已验证，增长可复制快速增长型扩张空间多大？新市场能否复制？PE是否低于或接近增长率？
利润主要由价格、库存、产能、商品周期决定周期型周期处在底部、上行段还是顶部？低PE是不是周期顶部陷阱？
公司当前有严重问题，关键在能不能活下来困境反转型现金能撑多久？债务是否可控？亏损源是否被处理？
市值明显低于现金、地产、牌照、子公司等资产扣债后价值隐蔽资产型资产是否真实？债务扣除后还剩多少？价值何时释放？

混合类型很常见。处理原则是：先找主导收益来源，再按主导类型估值和追踪。例如一家快速增长公司如果增长耗尽，可能迁移为稳定增长型；一家困境反转公司如果已经活过来，也要重新分类，不能继续用“反转弹性”解释它。

3.2 六类公司深度对比

维度缓慢增长型稳定增长型快速增长型周期型困境反转型隐蔽资产型
典型增速2%-4%10%-12%20%-25%大幅波动亏损到盈利不一定增长
常见合理PE低PE，股息为主约10-14倍视增长而定PE应与增长匹配不能机械看PE常无稳定PE看资产折价
你买的是什么股息和低预期稳健复利可复制增长周期上升段重生弹性被忽视财富
核心问题股息安全吗？好公司是否买贵？增长还能持续多久？周期位置在哪里？会不会先死？资产能否兑现？
最大误判用十倍股预期买股息股好公司买太贵单点成功当全国复制PE低时误以为便宜跌多了就以为会反弹只看账面不扣债务
卖出逻辑股息吸引力下降或涨30%-50%PE过高或增长放缓增长证伪或估值透支周期顶部信号出现转危为安后重分类资产被发现或价值被侵蚀

3.3 六类公司逐项手册

缓慢增长型：不要用十倍股期待买股息股

缓慢增长型不是坏公司，但它的题型不是成长，而是股息和低预期。

买入前看：

  1. 长期股息记录是否稳定。
  2. 股息是否持续提高。
  3. 派息率是否过高。
  4. 现金流是否覆盖股息。
  5. 公司是否缺乏新增长点。

卖出信号：

  1. 上涨30%-50%，但没有新增长逻辑。
  2. 连续两年市场份额流失。
  3. 削减研发、吃老本。
  4. 收购无关业务导致资产负债表恶化。
  5. 即使股价低，股息收益率仍没有吸引力。

稳定增长型：价格纪律比公司质量更重要

稳定增长型的错误通常不是买到烂公司，而是好公司买贵了。林奇对这类公司的预期常常是30%-50%的阶段性收益，而不是十倍股。

买入前看：

  1. PE是否低于或接近公司历史水平。
  2. PE是否低于或接近同类公司。
  3. 长期收益增长是否仍稳定。
  4. 经济衰退中是否相对抗跌。
  5. 主营业务是否被多元化稀释。

卖出信号：

  1. 股价线超过收益线。
  2. PE远超正常水平或同业水平。
  3. 管理层和董事过去一年没有买入。
  4. 占利润较大比例的业务容易受经济衰退冲击。
  5. 增长率放缓且成本削减空间有限。

快速增长型：林奇最喜欢，但必须证明可复制

快速增长型不是“行业很热”，而是“公司有一套已经验证、还能继续复制的增长机器”。林奇尤其喜欢低增长行业中的快速增长公司，因为竞争少、预期低、增长更容易留下利润。

最佳阶段不是创业期，而是第二阶段复制扩张期：

阶段状态投资含义
第一阶段产品或门店模型还在证明想象空间大，失败率高
第二阶段模式已证明，扩张空间仍大林奇式投资最适合介入
第三阶段增长接近饱和，需要新故事容易高估，PE可能压缩

买入前看：

  1. 公司已经盈利，资产负债表稳健。
  2. 收益增长约20%-25%，过高反而要怀疑可持续性。
  3. 成功模型已经在多个市场验证。
  4. 仍有足够扩张空间。
  5. PE接近或低于增长率。
  6. 机构持股和分析师关注还不拥挤。
  7. 产品或服务具有复购、利基或低竞争环境。

卖出信号：

  1. 同店销售下降。
  2. 新开分店表现差。
  3. 高级管理人员和核心员工离职去竞争对手。
  4. 公司开始密集路演并被机构热捧。
  5. PE达到30倍，但未来两年乐观增长只有15%-20%。
  6. 增长空间接近饱和。

周期型：不要被低PE欺骗

周期股的利润最好时，PE往往最低；利润最差时，PE反而可能最高甚至没有PE。研究周期股，核心不是“便宜”，而是周期位置。

买入前看：

  1. 行业库存是否下降。
  2. 商品价格是否企稳或上涨。
  3. 产能利用率是否改善。
  4. 竞争对手是否退出。
  5. 新产能是否被控制。
  6. 终端需求是否恢复。
  7. 公司成本位置是否有竞争力。

卖出信号：

  1. 成本上升。
  2. 工厂满负荷运行且公司开始扩产。
  3. 存货不断增加且处理不掉。
  4. 大宗商品价格下跌。
  5. 期货价格低于现货。
  6. 新竞争者进入并降价。
  7. 最终需求下降。

困境反转型：先证明不会死，再谈上涨空间

困境反转股最诱人的是弹性，最危险的是公司可能先死。跌幅不是买入理由，生存时间才是核心。

常见亚型：

  1. 外部救助型。
  2. 问题没预想严重型。
  3. 破产母公司中含优质子公司型。
  4. 剥离亏损业务、重组、削减成本型。
  5. 重新聚焦主业型。

买入前看：

  1. 现金能撑多久。
  2. 短债和长期债结构如何。
  3. 债权人是否能立刻逼债。
  4. 亏损源是否被关闭、剥离或重组。
  5. 主业是否出现修复证据。
  6. 是否有资产可卖，卖价是否可信。
  7. 是否存在养老金、诉讼、担保等隐藏义务。
  8. 是否可能通过增发严重稀释老股东。

卖出信号：

  1. 公司已经成功转危为安，需要重新分类。
  2. 连续下降的债务重新上升。
  3. 存货增长速度是销售增长速度的两倍。
  4. PE相对预期增长过高。
  5. 最强业务过度依赖一个正在放缓的大客户。

隐蔽资产型：账面价值只是线索，真实价值才有用

隐蔽资产可能是现金、地产、矿权、牌照、媒体资产、税损、子公司股权、品牌或专利。难点是：资产存在，不等于普通股东能拿到。

买入前看：

  1. 资产真实价值是多少。
  2. 资产是否能出售或释放。
  3. 债务、税费、少数股权要扣掉多少。
  4. 管理层是否会继续乱收购或增发。
  5. 是否有并购者、激进投资者、分拆、回购等催化剂。
  6. 市场是否真的没有识别这部分价值。

卖出信号：

  1. 被收购或资产价值被市场充分发现。
  2. 公司发行超过总股本10%的新股去做多元化。
  3. 资产实际售价低于预期。
  4. 税率变化削弱税损价值。
  5. 机构持股大幅上升，冷门优势消失。

3.4 类型迁移

同一家公司会改变类型。你买入时的题型，可能不是持有三年后的题型。

原类型可能迁移到触发条件投资动作
快速增长型稳定增长型规模变大，扩张空间下降，增长率回落降低增长预期，重算合理PE
快速增长型困境反转型盲目扩张、负债上升、新店失败、现金流恶化先看生存，不再用成长股逻辑持有
稳定增长型缓慢增长型主业成熟，增长接近GDP，股息成为主要回报看股息安全性和估值
稳定增长型困境反转型多元恶化、主业受损、债务压力上升重新评估现金、债务、亏损源
周期型困境反转型周期低谷叠加高杠杆和管理失误不再只看周期复苏，先看债务期限
困境反转型稳定/周期/资产型危机解除，市场重新认识公司重新分类，按新类型估值

4. 质量验证：13条正面线索与6类排除对象

对应原书主题：第8章“13条选股准则”和第9章“我避而不买的股票”。

林奇喜欢的公司，不是宏大叙事公司，而是低关注、可理解、竞争少、财务好、管理层对股东友好的公司。

4.1 13条正面线索

这13条不是迷信冷门，而是在寻找三类特征：华尔街不理、竞争者不进、客户离不开。

#线索方法论含义典型案例/提示
1公司名字枯燥甚至可笑低关注、低预期Pep Boys、ADP
2业务枯燥竞争少、报道少瓶盖瓶塞、优惠券处理
3业务令人厌恶或压抑市场天然回避清洗油污、垃圾处理、殡葬服务
4从母公司分拆被机构忽略，管理层释放小贝尔、玩具反斗城
5机构持股少、分析师少发现前阶段小银行、保险、地方公司
6被谣言或偏见压低市场误解垃圾处理、赌场等曾被偏见压低
7处于低增长或冷门行业新进入者少低增长行业中的增长公司
8有利基或特许权利润留得住石料场、区域报纸、专利药、品牌
9产品反复购买收入持续性强药品、软饮料、剃须刀片
10是高技术产品的用户享受技术降本，而不是承担技术竞争用电脑降本的服务公司
11内部人士买入利益一致，股价可能低估多名管理层买入比单一高管更有力
12公司回购股票资本配置对股东友好回购减少股本，提高每股收益
13财务记录稳健故事有数字支撑收益、现金、负债、存货一起看

4.2 6类排除对象

#排除对象为什么危险现代化识别方式
1热门行业的热门股高竞争、高估值、好消息已定价高融资、高热度、高估值，增长必须连续超预期
2被吹捧成“下一个XX”的公司模仿者常晚到，质量不如原公司只比较商业模式，不接受称号
3多元恶化公司拿股东的钱买不懂业务并购频繁、主业分散、负债上升、协同不清
4小声耳语股故事诱人但没有收益和证据概念多、现金流弱、研发故事无法验证
5过度依赖大客户的供应商议价权弱，客户取消订单会致命单一客户占收入25%-50%要警惕
6名字花哨的公司包装掩盖普通业务先看业务和财务，不被名称、概念、缩写影响

筛选结论只允许三种：

  1. 进入深度研究：线索重要、分类清楚、质量过关。
  2. 继续观察：线索有意义，但关键事实还没出现。
  3. 直接排除：落入排除清单，或故事无法被财务支持。

5. 估值与财务：价格、增长、现金、负债

对应原书主题：第10章“收益，收益，还是收益”和第13章“一些重要的财务分析指标”。

估值只回答三个问题：

  1. 现在的价格贵不贵。
  2. 公司能不能撑到故事兑现。
  3. 增长有没有转成现金。

因此，买入前至少要同时看四件事：PE与增长率、现金、负债、现金流。好故事如果没有收益和现金支持，就是想象；好公司如果价格太高，也会变成坏股票。

5.1 PE与增长率

基础规则：

合理PE ≈ 收益增长率

使用方式：

情况判断
PE约等于增长率大致合理
PE只有增长率一半可能有吸引力
PE是增长率两倍风险很高
极高PE要求极高且长期的盈利增长，通常很难兑现

加入股息后的粗略判断：

(长期收益增长率 + 股息收益率) / PE
结果判断
小于1不理想
约1.5不错
大于等于2很有吸引力

5.2 估值速算表

这张表不是机械买卖规则，而是压力测试起点。

增长率大致合理PE可能便宜明显昂贵
5%5-7<5>10
10%10-12<10>20
15%15-18<15>30
20%18-22<18>40
25%22-28<22>50

高PE灾难的共同点不是“公司完全没价值”，而是价格要求未来过于完美。EDS、宝丽来、雅芳这类案例都说明：好公司如果用离谱价格买入，也会变成高风险股票。

5.3 现金与负债

现金不是装饰，尤其对困境反转型、周期型和隐蔽资产型很关键。

净现金 = 现金 + 有价证券 - 长期负债
每股净现金 = 净现金 / 流通股数
实际买入价 = 股价 - 每股净现金
实际PE = 实际买入价 / 每股收益

负债分析必须问：

  1. 总负债多少。
  2. 短债还是长期债。
  3. 债权人是否能随时要求还款。
  4. 经营现金流能否覆盖利息和到期债务。
  5. 困境公司能撑多久。

5.4 12项财务指标速查

指标主要用途危险信号
产品/业务占比判断线索是否足够重要好产品占比太小
PE判断价格要求PE远高于增长
现金头寸判断安全垫现金被乱收购消耗
负债结构判断生存时间短债压力、再融资困难
股息判断低增长/稳定股回报派息率过高
账面价值判断资产保护资产卖不出、存货贬值
隐蔽资产判断市场遗漏债务和税费吃掉价值
现金流判断利润质量盈利增长但现金流差
存货判断需求和周期存货增速高于销售
养老金/或有负债判断隐藏义务报表外压力
增长率判断收益来源增长不可持续
税后利润率判断成本优势行业好时利润率不升，行业差时扛不住

5.5 增长率来自哪里

收益增长通常来自五类来源：

  1. 削减成本。
  2. 提高售价。
  3. 开拓新市场。
  4. 在原市场卖出更多产品。
  5. 重振、关闭或剥离亏损业务。

你要判断的是：增长来自一次性动作，还是可持续机制。林奇更偏好20%-25%这种可持续增长，而不是看起来高得离谱、但很难长期维持的增长。

6. 信息核查：用资料验证公司逻辑

对应原书主题：第12章“如何获得真实的公司信息”。

信息核查的目的不是收集更多观点，而是找事实来证明或推翻自己的买入故事。

6.1 核查对象

核查对象要回答的问题主要资料容易犯的错
信息披露文件公司自己披露了哪些事实，管理层如何解释业务变化年报、半年报、季报、临时公告、招股书、电话会/业绩说明会纪要把管理层叙事当结论，不回到数据验证
三张财务报表利润是否真实，现金是否流入，资产负债表是否安全利润表、资产负债表、现金流量表，以及附注把报表和披露文件混为一层；只看收入增长，不看现金和负债
分部与经营数据线索影响的业务是否足够重要，增长来自哪里分部收入/利润、产品占比、地区数据、门店数据、客户结构好产品占比太小，却当成公司整体变化
经营现场产品、服务、渠道、客户是否真的变好门店、用户评价、渠道反馈、客户访谈把个别体验当成整体趋势
行业变量供需、价格、库存、竞争是否支持公司行业数据、价格数据、竞品公告、产能变化只看行业热度，不看公司位置
管理层行为管理层是否站在股东一边回购、增发、并购、分红、内部人买卖听管理层叙事，不看资本配置结果
市场定价好消息是否已经被充分定价PE、增长率、机构持股、分析师覆盖、股价预期好公司买成坏价格

6.2 不要问股价，问事实

不要问：“为什么股价下跌？” 应该问：

  1. 收入增长来自价格、销量、新产品、渠道，还是并表？
  2. 毛利率、费用率、库存、回款、客户集中度有没有恶化？
  3. 某个产品或业务线对收入和利润影响多大？
  4. 扩张靠资本开支、加盟、并购、提价，还是复购？
  5. 单位经济模型是否成立？
  6. 短债何时到期，现金流能否覆盖？
  7. 是否需要再融资或增发？
  8. 新竞争者、替代技术、价格战有没有改变？
  9. 我的投资假设哪里可能错？

6.3 年报阅读顺序

  1. 现金和有价证券。
  2. 长期债务和短期债务。
  3. 净现金头寸。
  4. 流通股数量是否减少。
  5. 10年财务数据。
  6. 收益是否持续增长。
  7. 股息是否稳定。
  8. 存货是否异常。
  9. 养老金、诉讼、担保等或有负债。
  10. 资产价值是否被低估。

7. 买入：两分钟投资故事

对应原书主题：第11章“下单之前沉思两分钟”。

两分钟投资故事不是漂亮文案，而是“真正看懂公司”的检验。可以把它理解为段永平所说“看懂一个公司”的口头版本：如果你真的看懂了，就应该能用两三分钟讲清楚公司靠什么赚钱、为什么未来会更好、价格是否合理、哪些事实会证明你错了。

讲不清楚，就不该买。

7.1 通用八问

  1. 我为什么注意到这家公司？
  2. 它属于六类公司中的哪一种？
  3. 它靠什么提高收益？
  4. 这个增长、反转、周期或资产故事需要哪些事实成立？
  5. 当前估值要求它未来做到什么？
  6. 关键风险是什么？
  7. 买入后我要跟踪哪些指标？
  8. 什么情况下我会卖出或换仓？

7.2 按类型写故事

类型两分钟故事重点
缓慢增长型股息是否安全，为什么当前价格仍有吸引力
稳定增长型增长是否稳定，PE是否合理，是否有防御性
快速增长型成功模式如何复制，扩张空间还有多大，PE是否透支
周期型行业处在周期哪里，供需、库存、价格如何变化
困境反转型为什么不会死，哪些事实说明它正在活过来
隐蔽资产型资产值多少钱，债务扣多少，何时可能释放

不能通过两分钟故事的情况：

  1. 只会说“它会涨”。
  2. 只会说“产品很好”。
  3. 只会说“别人推荐”。
  4. 只会说“跌了很多”。
  5. 只会说“行业很热”。
  6. 说不清失败条件和卖出条件。

8. 买后持续追踪：按公司类型建立观察指标

对应原书主题：第14章“定期重新核查公司分析”、第15章“股票分析要点一览表”、第16章“构建投资组合”。

买入后不是每天问股价，而是问故事发生了什么差分。

看到的变化错误反应正确追问
股价上涨我对了，所以继续拿盈利、扩张、估值是否仍支持原故事？
股价下跌我错了，所以止损基本面变坏，还是价格给了机会？
公司并购规模更大就是更好是否相关？价格是否过高？资产负债表是否恶化？
快速扩张增长越快越好新店/新市场是否复制成功？资金和管理是否跟得上？
困境股反弹反转开始了，继续等十倍债务是否下降？问题是否解决？是否应重新分类？

8.1 类型化追踪仪表盘

类型追踪指标观察频率行动规则
快速增长型同店销售、新店表现、扩张空间、PE/增长率、机构拥挤度每季增长强且估值合理，持有或加仓；同店下滑或新店失败，降级研究
周期型库存、商品价格、产能、订单、竞争者进入高频/每季供需改善时持有；库存和价格转坏时卖出
困境反转型现金消耗、债务期限、成本削减、亏损业务、客户恢复每季/事件驱动生存事实改善才持有；债务恶化或增发风险上升，卖出
隐蔽资产型资产价值、债务、资产出售、并购者、资本配置每季/事件驱动资产释放前持有；债务和增发侵蚀价值，卖出
稳定增长型收益增长、PE、核心业务、成本削减空间每季增长稳定且PE合理持有；PE过高或增长放缓，换仓
缓慢增长型股息、派息率、现金流、市场份额每半年股息安全才持有；股息吸引力消失或业务恶化，卖出

8.2 监控失败案例：得克萨斯航空

林奇曾在得克萨斯航空上经历过一个典型错误：低价买入后股价大涨，后来在高位追加，却忽视了行李丢失、航班混乱、员工不满、劳资纠纷等基本面恶化信号。股价从高位大幅回落。

教训：不要被股价上涨冲昏头脑。基本面变坏就是卖出信号，和你赚了多少无关。

9. 卖出/换仓：故事破裂、估值过高、机会更好

对应原书主题：第17章“买入和卖出的最佳时机选择”、第18章“12种危险说法”、第20章“长期投资”。

林奇的卖出原则不是止盈止损，而是：如果你知道为什么买，就应该知道什么时候卖。

9.1 卖出总顺序

顺序问题触发动作
1买入故事还成立吗？不成立就卖出
2公司类型是否已经迁移？重新分类，按新类型重估
3估值是否严重透支未来？减仓、卖出或换到更好机会
4是否出现更强、更便宜、更可验证的机会？换仓
5以上都没有触发继续持有并定期复查

林奇反对以下卖出理由：

  1. 股价涨了一倍，所以卖。
  2. 股价跌了20%，所以卖。
  3. 大盘可能下跌，所以卖。
  4. 等反弹到买入价再卖。
  5. 这只股票太无聊，所以卖。
  6. 别人说见好就收，所以卖。

9.2 按类型卖出表

类型核心卖出信号
缓慢增长型上涨30%-50%、股息吸引力下降、市场份额流失、多元恶化
稳定增长型PE远高于历史/同业、增长放缓、利润部门受衰退冲击
快速增长型扩张放慢、同店销售下降、空间耗尽、估值远高于增长、机构过度拥挤
周期型存货增加、商品价格下跌、产能扩张、需求下降、竞争者进入
困境反转型已经转危为安、债务重新增加、库存异常、反转故事结束
隐蔽资产型被收购、资产价值被发现、债务侵蚀资产、发行股票做多元化

9.3 卖出前的最后七问

  1. 如果今天没有持有，我还会用今天的价格买入吗？
  2. 我卖出是因为基本面，还是因为股价？
  3. 买入理由是否已经失效？
  4. 是否只是受到市场噪音影响？
  5. 如果股价再跌25%，我愿不愿意加仓？
  6. 如果不愿意加仓，为什么还持有？
  7. 卖出后资金要去哪里，新机会是否更好？

10. 组合管理与心理护盾

对应原书主题：第16章、第18章、第19章、第20章。

10.1 收益目标和持股数量

  1. 市场长期平均约9%-10%。
  2. 如果自己选股，长期目标应争取12%-15%，否则指数基金更省力。
  3. 25%-30%的长期年化预期不现实，会诱导冒险和频繁交易。
  4. 小组合通常3-10只比较合理，但关键不是数量，而是你是否真正研究过。
  5. 为了分散而买不懂的股票，是愚蠢的多元化。

林奇的组合不是平均铺开，而是把资金从故事变弱的公司转移到故事更强的公司。不要“拔掉鲜花浇灌野草”：不要卖出基本面变好的赢家，去补仓基本面变差的输家。

10.2 类型配置

类型在组合中的作用
稳定增长型降低组合波动
快速增长型提供长期复利
周期型提供阶段弹性
困境反转型提供非市场相关机会
隐蔽资产型提供资产重估机会

配置不是机械比例，而是根据你能理解、能验证、价格合理的机会调整。

10.3 12种危险说法

危险说法纠正
跌这么多，不可能再跌了可以跌到零
总能知道什么时候到底底部只有事后才知道
涨这么高，不可能再涨好公司没有人为上限
股价只有几美元，亏不了多少跌到零都是亏100%
最终会涨回来烂公司可能永远回不来
黎明前总是最黑暗有时黑暗之后更黑
反弹到某价位我再卖心理价位不是基本面
保守股不会大跌蓝筹也会大跌
等太久，不可能涨了基本面增长可能多年后才反映
我错过了，损失很大没买不亏钱
我要抓住下一只同类股“下一个XX”常是陷阱
上涨证明我对，下跌证明我错股价短期不能证明基本面

10.4 远离复杂工具

林奇把期权、期货、卖空放在长期投资体系之外。原因不是这些工具永远不能赚钱，而是它们会把投资者从研究企业拉回预测价格。

工具核心问题
期权到期日会强迫你在错误时间做判断
期货杠杆高，波动放大，容易变成短线投机
投资组合保险机械卖出会在大跌中放大踩踏
卖空理论收益有限，亏损可能无限，且泡沫能持续很久

11. 三个高频问题

11.1 什么时候卖？

不是涨了卖，不是跌了卖，而是：

  1. 故事结束就卖。 买入理由消失，不管盈亏都卖。
  2. 估值极端就卖。 股价已经提前透支未来增长。
  3. 类型变化就重估。 困境反转成功后，不再用反转逻辑持有。
  4. 有更好机会就换。 卖出不是逃离股市，而是在更好基本面之间配置资金。

最实用的一句话：买入时写下故事，卖出时只检查故事。

11.2 要不要做宏观分析？

要，但位置要放对。

能做不能做
看市场整体PE，判断风险温度用市场PE直接决定清仓
看上涨/下跌家数比，判断市场广度用指数涨跌解释个股对错
看利率、油价、汇率对行业利润的影响用宏观新闻替代公司研究
看周期行业供需、库存、价格把所有股票都当宏观交易

林奇式宏观分析的标准是：能否落到收入、成本、需求、估值、融资、库存这些公司变量上。

11.3 困境反转型怎么提高确定性？持续下跌时先等待吗？

困境反转要分三种状态：

状态特征动作
只是跌得多没有债务改善、没有亏损源处理、没有主业恢复继续观察，不买
生存已大概率确定现金足够，短债压力可控，有资产可卖，债权人不逼债可以进入深度研究
反转事实已经出现成本下降、亏损业务剥离、订单/客户恢复、债务下降才进入买入区间

持续下跌时不是机械等待，也不是越跌越买，而是盯三类事实：

  1. 生存事实： 现金、债务期限、债权人压力。
  2. 修复事实： 成本削减、亏损源剥离、主业恢复。
  3. 股东事实： 是否增发、资产是否被贱卖、债务是否侵蚀普通股。

如果这三类事实没有改善，股价再低也只是更便宜的危险。

13. 全书最重要的14个见解

  1. 业余优势不是比机构聪明，而是有机会更早看到真实经营变化。
  2. 常识不是拍脑袋，而是到真实世界观察，再用简单数字验证。
  3. 产品好、公司好、股票好，是三件不同的事。
  4. 六种公司类型不是标签，而是问题生成器。
  5. 十倍股多来自第二阶段复制扩张，不是第一阶段纯想象。
  6. 低增长行业中的增长公司，常比热门行业中的热门公司更好。
  7. 困境反转的核心不是跌幅，而是生存时间。
  8. 周期股最危险的时刻，经常是PE看起来最低的时候。
  9. 资产价值只有扣掉债务、流动性和管理层行为后才算数。
  10. 估值不是公式，而是对故事的压力测试。
  11. 买入和卖出是同一件事：买入写故事，卖出查故事。
  12. 宏观不是不用，而是必须翻译成公司变量。
  13. 组合管理的核心是让正确继续正确，不要拔掉鲜花浇灌野草。
  14. 心理防线不是鸡汤，而是防止系统被情绪覆盖。
